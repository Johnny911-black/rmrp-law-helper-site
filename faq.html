Частые вопросы

Установка, биндер, патруль и безопасность — короткие ответы.

Это официальное приложение RMRP?

Нет. RMRP Law Helper — неофициальный fan-tool для игроков. Не связан с администрацией сервера.

Безопасно ли скачивать? Антивирус ругается

Приложение не подписано платным сертификатом — Windows SmartScreen может предупредить. Скачивайте только с официальной страницы «Скачать» или GitHub Releases проекта.

Собираете ли вы данные?

Нет. Нет регистрации, аналитики и телеметрии. Сайт тоже без трекеров. Настройки хранятся локально на вашем ПК.

Бинды не отправляются в чат

Запустите приложение от имени администратора, если GTA с admin-правами. Переключите фокус на игру. Проверьте в настройках клавишу чата (T, Y, F4). В разделе биндера — статус Native hotkeys должен быть 2/2 или больше.

Нужен ли интернет?

Для первого запуска и обновления законов — желательно. После кэширования справочник работает офлайн.

Как обновить приложение?

Скачайте новый архив из GitHub Releases и замените старую папку или файл. Настройки и профили сохранятся в userData Windows.
