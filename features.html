Возможности

Подробный обзор модулей приложения — от поиска по УК до автодокладов на патруле. Нажмите на скриншот для увеличения.

Законодательство

Все законы с форума

Актуальное законодательство RMRP с официального форума: УК, КоАП, УПК, Конституция, трудовой и другие кодексы. Данные кэшируются локально и обновляются с форума — можно работать офлайн после первой загрузки.

  • Fuzzy-поиск: «ст 105», «УПК 186», текст, наказание
  • Вкладки документов — закрепите нужные кодексы
  • Режим «Полный текст» с подсветкой совпадений
  • Карточки статей с наказаниями и картинками
  • Перекрёстные ссылки между статьями
  • Закладки, отдельное окно, заметка к статье
Правила сервера

Все правила сервера с форума

Актуальные правила RMRP с официального форума: категории, документы, разделы и пункты. Тот же поиск и закладки, что и в законах — переключение одной кнопкой в sidebar.

  • Иерархия: категория → документ → раздел → пункт
  • ПКМ по категории — полный текст, отдельное окно
  • Общие закладки и заметки для laws + rules
  • Сохранение позиции при переключении режимов
Статьи

Карточки с наказаниями

Статьи сгруппированы по главам. Видны наказания, звёзды розыска, части статьи. Закрепление в профиль, добавление в свою подборку, открытие в отдельном окне.

  • Группировка по главам и кодексам
  • Закрепление главы или отдельной статьи
  • Умный поиск с подсветкой совпадений
Биндер

Макросы в чат по hotkey

Создайте текстовые бинды с горячими клавишами — отправка напрямую в чат GTA через native SendInput. Несколько профилей биндов для разных ролей и персонажей.

  • Режимы «В чат» (Enter) и «Только вставить»
  • Плейсхолдеры: {fio}, {rank}, {org}, {time}, {date}
  • Панель биндов — мини-окно со списком поверх игры
  • Экспорт/импорт профиля кодом RMRP-BINDS-v1
  • Работает в фоне при скрытом главном окне
  • Native / hybrid / electron режимы hotkey
Дежурство

Пост и патруль

Для ГИБДД и похожих фракций: автоматические доклады с званием и ФИО из профиля, учёт пола персонажа в формулировках, таймер до следующего доклада.

  • Старт смены — доклад в чат автоматически
  • Периодический доклад по интервалу (например, 10 мин)
  • Hotkey доклада и поддержки
  • Мини-overlay дежурства с таймерами
  • Звуковое напоминание «пора докладывать»
  • Завершение смены — финальный доклад
Workspace

Закладки и заметки

Единый блокнот и закладки на статьи, главы и свои подборки. Несколько профилей workspace — для разных персонажей.

  • Свои закладки-подборки для laws и rules
  • Выбор профиля при закреплении статьи
  • Исключение статей и частей из подборки
  • Экспорт кода RMRP-WORKSPACE-v1
  • Импорт в новый профиль без потери текущего
Заметки

Блокнот для смены

Общий блокнот для всех разделов: номера, контакты, напоминания, выдержки из статей. Закрепление заметки во вкладку, экспорт вместе с закладками через workspace.

Карты

Налёты и локации

Справочные карты RMRP: налёты, поставки, локации. Можно добавлять свои картинки для быстрого просмотра поверх игры.

Overlay

Поверх игры

Окно always on top, изменение размера, ghost-режим (клики в игру), глобальный hotkey показа/скрытия. Отдельные окна для статей.

  • F9 — показать/скрыть (настраивается)
  • Ghost на главном окне — только кнопка в шапке
  • Ghost на мини-панелях — hotkey + кнопка 👆
  • Клавиша чата GTA и hotkeys биндера в настройках
  • 6 цветовых тем интерфейса
Онлайн

Трекер времени в GTA

Отслеживание процесса GTA5.exe: время сегодня, текущая сессия, streak дней подряд, графики за неделю.

  • Статус «В игре» / «GTA не запущен»
  • График активности по часам
  • Список сессий за день
